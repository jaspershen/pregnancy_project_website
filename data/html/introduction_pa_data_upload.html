Project name

The project name is required. For example, 20180919test.

Metabolite type

MetFlow supports KEGG ID and m/z values of peaks.

Metabolite list

If you select KEGG ID in Metabolite type, you should paste the KEGG ID of dysregulated metabolites. If you select Peak (m/z) in Metabolite type, you should paste the m/z values of dysregulated peaks.
