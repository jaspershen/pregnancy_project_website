How to begin?

1. Select data and load data

1.1. Demo data
Click Upload data, and select Data from Demo data, then click Load data.
1.2. Local data
Click Upload data, and select Data from Local data, then click Load data.
1.3. Previous step data
Click Upload data, and select Data from Previous step, then the data information of data in other projects will appear, then change the name of project, and click Load data.

2. Set parameter and process

Click Parameter setting, and set parameters, then click Submit to process.
